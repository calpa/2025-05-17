# 什麼是 Google Apps Script？

- Google 官方雲端自動化腳本
- 可操控 Docs、Sheets、Forms、Gmail 等
- 免安裝、免費、雲端運作

---

# 什麼是 ChatGPT？

- AI 文字生成工具
- 可用於自動產生文件、郵件、表單內容
- 可與 Apps Script API 串接
